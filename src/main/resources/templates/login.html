About application

Post-stroke patients require systematic rehabilitation to regain physical mobility and be able to function independently. TEREHAB makes it possible to get good results from home, which is especially important in the face of a global pandemic.

The application enables the patient to communicate with a specialist via an audio-video system. A specialist can create an individual training plan for each patient, using the database of available exercises. Moreover, the therapist can save notes on the results obtained, monitor the frequency of exercise or change the patient's training plan on an ongoing basis.
